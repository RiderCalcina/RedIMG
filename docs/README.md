# RedIMG - Modern Image Optimizer v3.5.3

**RedIMG** es una herramienta potente y ultra-rápida diseñada para la optimización y redimensionamiento masivo de imágenes. Enfocada en la eficiencia, ofrece procesamiento en paralelo y una calidad visual superior.

## ✨ Características Premium
- **Procesamiento en Paralelo**: Multi-hilo real que aprovecha hasta 4 núcleos para máxima velocidad.
- **Búsqueda Binaria de Calidad**: Algoritmo optimizado que encuentra el peso ideal en milisegundos.
- **Simetría Minimalista**: Diseño de ventanas gemelas alineadas para una experiencia de usuario balanceada.
- **Interfaz Pure Black Deep**: Estética basada en `ttkbootstrap` con fondo y acento negro absoluto.
- **Soporte de Carpetas**: Arrastra directorios enteros para una optimización automática en lote.

## 🚀 Inicio Rápido
1.  **Ejecución**: Inicia `RedIMG-v3.0.py`.
2.  **Acción**: Arrastra carpetas o imágenes al centro, o haz clic para abrir el explorador.
3.  **Resultado**: Las imágenes optimizadas se guardarán en la subcarpeta `/RedIMG`.
