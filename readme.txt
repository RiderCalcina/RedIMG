RedIMG v3.5.3 - Quality & Speed Pro
====================================

Tu herramienta definitiva para la optimización masiva de imágenes JPG con calidad profesional.

PUNTOS CLAVE:
------------
- Optimización Inteligente: Búsqueda binaria exacta para alcanzar 780KB con máxima fidelidad.
- Calidad Pro: Resolución de 1600px y codificación de color 4:2:0 para resultados nítidos.
- Turbo Batch: Procesamiento en paralelo real utilizando múltiples núcleos de CPU.
- Multiselección: El botón individual ahora permite elegir y procesar múltiples fotos a la vez.
- Seguridad RedIMG: Carpeta de salida automática que garantiza que tus originales nunca se tocan.
- Interfaz High-End: Tema "Dark Black" minimalista con animaciones optimizadas para bajo consumo.

INICIO RÁPIDO:
-------------
1. Ejecute RedIMG-v3.0.py (v3.5.3 estable).
2. Use "En Lote" para carpetas o "Individual" para uno o varios archivos específicos.
3. Encuentre sus fotos optimizadas en la subcarpeta (/RedIMG).

DOCUMENTACIÓN:
-------------
Consulte la carpeta (docs) para análisis técnico detallado, comparativas de calidad y guías de uso.
